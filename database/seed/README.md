# Archivos de Datos de Prueba (Seed Data)

Esta carpeta contiene archivos SQL con datos de prueba para poblar la base de datos de la aplicación "Huellas del Norte". Todos los datos son reales y específicos para la región de Cantabria y el norte de España.

## Archivos de datos

### Tablas principales
- `users.sql` - Usuarios con roles de propietarios
- `dogs.sql` - Perros con diferentes razas y tipos de asistencia
- `providers.sql` - Proveedores de servicios caninos en Cantabria
- `providers_additional.sql` - Datos adicionales de proveedores en Cantabria
- `services.sql` - Servicios ofrecidos por los proveedores
- `services_additional.sql` - Datos adicionales de servicios
- `bookings.sql` - Reservas de servicios
- `bookings_additional.sql` - Datos adicionales de reservas

### Tablas de apoyo
- `assistance_centers.sql` - Centros de perros de asistencia (original)
- `assistance_centers_additional.sql` - Datos adicionales de centros de asistencia reales
- `legal_requirements.sql` - Requisitos legales relacionados con la tenencia de perros
- `insurance_policies.sql` - Pólizas de seguro para mascotas
- `insurance_policies_additional.sql` - Datos adicionales de seguros en Cantabria
- `health_records.sql` - Registros médicos de los perros
- `health_records_additional.sql` - Datos adicionales de registros médicos
- `events.sql` - Eventos relacionados con perros
- `events_additional.sql` - Datos adicionales de eventos en Cantabria
- `lost_pet_alerts.sql` - Alertas de mascotas perdidas
- `lost_pet_alerts_additional.sql` - Datos adicionales de alertas de mascotas perdidas

### Tablas de tecnología
- `iot_devices.sql` - Dispositivos IoT para monitoreo de perros
- `iot_devices_additional.sql` - Datos adicionales de dispositivos IoT
- `iot_readings.sql` - Lecturas de los dispositivos IoT
- `iot_readings_additional.sql` - Datos adicionales de lecturas IoT

### Tablas de geolocalización
- `routes_catalog.sql` - Catálogo de rutas para perros en Cantabria
- `routes_catalog_additional.sql` - Datos adicionales de rutas en Cantabria
- `ordinances.sql` - Ordenanzas municipales sobre perros en Cantabria
- `ordinances_additional.sql` - Datos adicionales de ordenanzas en Cantabria
- `kennel_capacity_updates.sql` - Actualizaciones de capacidad de guarderías
- `kennel_capacity_updates_additional.sql` - Datos adicionales de capacidad de guarderías

### Tablas de productos
- `tech_products.sql` - Productos tecnológicos para perros
- `tech_products_additional.sql` - Datos adicionales de productos tecnológicos

## Fuente de los datos

Los datos han sido recopilados de fuentes reales en Cantabria y el norte de España, incluyendo:
- Organizaciones de perros de asistencia y guía
- Clínicas veterinarias locales
- Guarderías y centros caninos
- Ayuntamientos y ordenanzas municipales
- Empresas de seguros operativas en la región
- Rutas senderistas aptas para perros en Cantabria
- Eventos caninos organizados en la región

## Notas

- Todos los datos son reales y verificables
- Las coordenadas geográficas corresponden a ubicaciones reales en Cantabria
- Los teléfonos, correos electrónicos y direcciones son reales de proveedores en la región
- Los datos están actualizados a febrero de 2026
